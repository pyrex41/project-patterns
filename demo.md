# Building project-patterns: A Multi-Agent CLI

*2026-04-03T17:10:19Z by Showboat 0.6.1*
<!-- showboat-id: fed7c767-85c1-4bad-946c-7eab45e5cd19 -->

project-patterns is a Go CLI for maintaining a personal index of reference projects with tags and descriptions, designed for progressive disclosure in AI coding workflows. It was built entirely through a multi-agent AI pipeline: spec on grok.com, task decomposition via scud, review via scud heavy, and implementation via Claude Code orchestrating parallel Sonnet agents.

Let's verify it builds and walk through the commands.

```bash
go build -o project-patterns . && echo 'Build successful'
```

```output
Build successful
```

```bash
./project-patterns --help
```

```output
project-patterns is a CLI tool for indexing, searching, and syncing
reference projects used in AI-assisted coding workflows.

It lets you maintain a curated catalog of local directories and remote
repositories, tag them for retrieval, and pipe them into tools like showboat
for context injection.

Usage:
  project-patterns [command]

Available Commands:
  add         Add projects to the index
  completion  Generate the autocompletion script for the specified shell
  config      Show or edit configuration
  help        Help about any command
  list        List all indexed projects
  search      Search projects by tags and keywords
  showboat    Run showboat on a project
  sync        Clone or pull all git projects
  version     Print version information

Flags:
      --config string   path to config file (default: ~/.config/project-patterns/config.yaml)
  -h, --help            help for project-patterns

Use "project-patterns [command] --help" for more information about a command.
```

Add some projects to the index — the tool itself, plus a remote repo.

```bash
./project-patterns add local . --name "project-patterns" --tags go,cli,ai --desc "Personal project index for AI coding workflows"
```

```output
Updated project: project-patterns (project-patterns)
```

```bash
./project-patterns add repo https://github.com/pyrex41/scud --tags go,cli,agents --desc "Multi-agent task orchestration"
```

```output
Added project: scud (scud)
```
